🚀 QUICK START SUMMARY
================================================================================================

1. Install Node.js 18+ from https://nodejs.org/
2. Install Docker Desktop from https://www.docker.com/
3. cd C:\CRM\backend && npm install
4. cd C:\CRM\frontend && npm install
5. docker-compose up -d (start MySQL)
6. cd backend && npm run dev (terminal 1)
7. cd frontend && npm run dev (terminal 2)
8. Open http://localhost:5173
9. Login: admin@crm.com / admin123
